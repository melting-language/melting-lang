Example extension

Minimal loadable extension for the Melt interpreter. It registers three built-ins: exampleVersion(), exampleAdd(a, b), and exampleSqrt(x).

Documentation

  • Developer guide — how to write and build Melt extensions (contract, API, config, build steps).

Quick start

  • Build: cmake -B build && cmake --build build (from repo root).
  • Enable: extension = example in bin/melt.ini.
  • Run: bin/melt examples/extension_example.melt.
