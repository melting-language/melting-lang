# Melt syntax highlighter extensions

This folder holds editor extensions that add **syntax highlighting** (and related support) for the Melt language.

## Contents

| Folder | Description |
|--------|-------------|
| **vs_code_extension** | VS Code extension: grammar, language config, and how to install/package |

## VS Code

See [vs_code_extension/README.md](vs_code_extension/README.md) for installation and packaging. In short: open the `vs_code_extension` folder in VS Code and use “Install Extension from Location”, or run `vsce package` to build a `.vsix`.
